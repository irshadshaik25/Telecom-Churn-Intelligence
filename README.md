# Telecom-Churn-Intelligence
An end-to-end Machine Learning pipeline developed to predict customer churn in the telecom industry. Features a risk segmentation framework (High/Medium/Low) and an interactive Plotly dashboard to drive data-informed retention strategies.

# 📡 Telecom Customer Churn Intelligence
**An End-to-End ML Pipeline for Predictive Retention & Risk Segmentation**

## 📌 Project Overview
In the highly competitive telecom industry, retaining existing customers is 5x more cost-effective than acquiring new ones. This project develops a machine learning solution to identify "at-risk" customers and categorizes them into actionable risk tiers.

### 🚀 Key Performance Results
* **Final Model:** Logistic Regression (Optimized for Interpretability)
* **Accuracy:** 81.2%
* **ROC-AUC Score:** 0.84
* **Key Finding:** Month-to-month contract holders are 4x more likely to churn than those on long-term plans.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** Pandas, NumPy, Scikit-Learn, XGBoost
* **Visualization:** Plotly (Interactive Dashboards), Matplotlib, Seaborn
* **Environment:** Jupyter Lab / Anaconda
